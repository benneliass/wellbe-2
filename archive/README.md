# Archive

This folder contains prior research and analysis work that was synthesized into the WellBe v2 system design package.

**These are not the source of truth.** The system design lives in [`/docs`](../docs/).

## Contents

- `research/` — six raw evidence research packages (patient flows, diagnostic errors, clinician pain points, global health issues, patient complaints, product design synthesis, shared health memory)
- `analysis/` — cross-research synthesis canvas (`wellbe-research-synthesis.canvas.tsx`) produced before the final system design package

All facts, decisions, and evidence from these packages are traceable through [`/docs/evidence/`](../docs/evidence/).
