# Research evidence package

**Research topic:** Research B - missed signals, diagnostic errors, and late discovery cases

**Date packaged:** 2026-05-30

This archive packages the completed Research B workbook into a clean evidence archive. It separates source logs, extracted facts, quotes, cases/examples, pattern analysis, and product/system implications.

## How to use

1. Start with `07_archive/manifest.md` for counts and file overview.
2. Use `01_sources/sources_master.csv` to inspect sources.
3. Use `02_extracted_evidence/cases_or_examples.csv` and `evidence_items.csv` for row-level evidence.
4. Use `03_analysis/` and `05_implications/` for interpretation, clearly separated from extracted evidence.
5. The original completed workbook is preserved under `04_topic_outputs/tables/` and `01_sources/raw_sources/screenshots_or_exports/`.
