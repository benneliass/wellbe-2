# User Workflows

## Patient workflow

1. Create or update a Health Thread.
2. Enter what changed from normal.
3. Add timeline, daily impact, prior care, main fear, and desired help.
4. Review visit summary after the encounter.
5. Mark missing/wrong/unaddressed items.
6. Track pending results, referrals, and follow-up obligations.

## Clinician workflow

1. Open one-page Health Thread summary.
2. Review patient words, timeline, relevant visits/tests, unresolved items, and pending actions.
3. Check source links and uncertainty states.
4. Update clinical reasoning, plan, and follow-up ownership.
5. Route correction requests or safety concerns as needed.

## Admin/care-coordination workflow

1. Review pending results and referrals.
2. Identify unclear owners or overdue items.
3. Escalate repeat visits or unresolved check-ins according to protocol.
