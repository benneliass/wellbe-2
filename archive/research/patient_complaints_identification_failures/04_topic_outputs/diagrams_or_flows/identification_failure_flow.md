# Identification failure flow

Patient concern -> intake/history -> triage/vitals -> clinician assessment -> tests/referral -> interpretation -> diagnosis discussion -> discharge/follow-up -> repeat visit/escalation

Common loss points:

1. Patient concern not captured.
2. Red flag normalized or dismissed.
3. Test result treated as closure rather than evidence.
4. Abnormal result lacks owner.
5. Referral rejected or routed to wrong specialty.
6. Discharge closes encounter but not unresolved problem.
7. Repeat visits are not linked into a longitudinal pattern.
