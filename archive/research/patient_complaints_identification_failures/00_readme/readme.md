# Research evidence package

**Research topic:** Research C - Patient complaints, raw quotes, reviews, and experience data about health-system identification failures

**Date packaged:** 2026-05-30

This archive packages the completed Research C evidence about real patient complaints, reviews, quotes, and experience data related to health-system failures or successes in identifying health issues.

## How to use this archive

1. Start with `04_topic_outputs/main_report.md` for a readable summary.
2. Use `01_sources/sources_master.csv` to see every source and URL.
3. Use `02_extracted_evidence/quotes.csv` for the quote bank.
4. Use `02_extracted_evidence/cases_or_examples.csv` for individual documented stories and incidents.
5. Use `03_analysis/` for patterns, confidence levels, gaps, and contradictions.
6. Use `05_implications/` for practical and product/system implications.

Raw source material is separated from processed evidence. Because live web pages were not downloaded during packaging, each raw-source folder contains small source-summary Markdown files with URLs and context.
