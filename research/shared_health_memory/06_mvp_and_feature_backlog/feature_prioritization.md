# Feature Prioritization

## Best MVP candidates

1. Health Thread intake
2. Clinician one-page Health Thread summary
3. Pending result and referral loop tracker
4. Patient correction loop
5. Evidence traceability layer

These features are practical today and aligned with memory, continuity, and follow-up support.

## Near-term but integration-heavy

- Trend-over-noise patient-generated-data summarizer
- Repeat-visit and unresolved-symptom escalation view

These require workflow integration and careful alert design.

## Later / caution-needed

- Clinical decision memory module
- Cross-specialty pattern map

These are valuable but require stronger governance, careful UX, validation, and legal/regulatory review.

## Avoid as early promise

- Autonomous diagnosis
- Rare-disease prediction as a direct claim
- Black-box risk scoring
- Raw wearable data dumping
