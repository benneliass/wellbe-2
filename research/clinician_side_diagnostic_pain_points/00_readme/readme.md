# Research evidence package

**Research topic:** Research D — clinician-side pain points in identifying health issues early

**Packaging date:** 2026-05-30

This archive packages the completed Research D work on clinician-side pain points in identifying health issues early. It separates source tracking, extracted facts, short quotes, cases/examples, metrics, analysis notes, and product/system implications.

## How to use

1. Start with `07_archive/manifest.md` for counts and included files.
2. Use `01_sources/sources_master.csv` to see every source and URL/identifier.
3. Use `02_extracted_evidence/evidence_items.csv` as the broad evidence table.
4. Use the more specific CSVs for facts, quotes, cases/examples, reviews/complaints, and metrics.
5. Read `04_topic_outputs/main_report.md` for a narrative summary.

Raw source files were not downloaded during the original research run. The `raw_sources/` folders therefore contain source-summary Markdown files with URLs and reasons raw files were not saved.
