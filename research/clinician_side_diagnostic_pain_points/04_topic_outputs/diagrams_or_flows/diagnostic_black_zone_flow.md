# Diagnostic black-zone flow

before visit → intake/online form → registration → nurse triage → first clinician assessment → history/exam → labs/imaging ordered → result interpretation → diagnosis discussion → referral/discharge → follow-up/repeat visit → specialist review

Common loss points:

- patient story compressed into form fields
- prior history not surfaced
- abnormal result buried in alerts
- referral lacks owner
- safety-net plan unclear
- repeat visits not connected
- handoff loses pending tasks
