# Source notes

The package uses a mixed evidence base: high-confidence institutional reports, health-system profiles, peer-reviewed studies, patient-safety investigations, ombudsman complaint reports, and implementation case studies.

Most sources are high quality for system/workflow claims. Medium-quality sources were used where they provided important cultural or implementation context but were less comprehensive or more case-specific.

Raw source files are mostly summary stubs, not full source captures. Each stub includes source title, URL, access date, short summary, and reason the raw source was not saved.
