# Research evidence package

**Research topic:** Research E — Global, cultural, and resource-level comparison of health issue identification workflows

**Date packaged:** 2026-05-30

This archive packages the completed Research E work on how health issue identification differs across cultures, continents, care settings, and resource levels.

## How to use this archive

Start with `04_topic_outputs/main_report.md` for the readable report. Then use `01_sources/sources_master.csv` to inspect sources, `02_extracted_evidence/` for structured evidence, and `03_analysis/` plus `05_implications/` for interpretation and design implications.

Raw/source material is separated from cleaned evidence. In many cases the package contains source-summary stubs rather than full source captures.
