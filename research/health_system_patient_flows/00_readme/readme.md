# Research evidence package

**Research topic:** Real health-system patient flows for identifying health issues  
**Date packaged:** 2026-05-30

This archive packages the completed Research A workbook into a clean evidence archive. It contains source metadata, extracted evidence tables, quotes, cases/examples, black-zone matrices, care-setting flow notes, interpretation notes, and the original workbook.

## How to use
1. Start with `07_archive/manifest.md` for counts and package contents.
2. Use `01_sources/sources_master.csv` to trace every source.
3. Use `02_extracted_evidence/evidence_items.csv` for the broad master evidence table.
4. Use `04_topic_outputs/main_report.md` for a readable synthesis.
5. Use `04_topic_outputs/tables/health_system_patient_flows_research_a.xlsx` if you prefer the original workbook format.

Facts, quotes, cases, and interpretation are separated into different folders/files.
