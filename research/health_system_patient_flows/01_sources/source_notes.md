# Source notes

The source set mixes clinical guidelines, workflow tools, public-health reports, patient-safety investigations, peer-reviewed studies, advocacy reports, and public patient-experience materials. High-confidence workflow facts usually come from official clinical/public-health sources. Patient voice and complaint sources provide lower-generalizability but high-context evidence about lived failure modes.

Raw source files were not re-downloaded during packaging. Each source has a placeholder Markdown file with URL, source type, setting, and summary.
