# Packaging notes

The completed workbook was treated as the authoritative completed research output. Workbook sheets were converted into structured CSVs and readable Markdown summaries. Raw web/PDF source content was not downloaded during packaging; instead, each source has a short source-summary Markdown file with title, URL, date/access information, and reason raw source was not saved.

Structured CSV rows include `research_topic` and `date_packaged` where required. Empty folders contain `.keep` files.
