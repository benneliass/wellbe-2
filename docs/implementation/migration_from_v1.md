# Migration from WellBe v1

## Keep unchanged

- Raw Context Vault
- Processing Pipeline
- Auth & Consent
- API Gateway
- Safety Engine requirement
- Evidence Ranking and Context Integrity principles

## Extend

- Concern Tracker -> Health Thread Engine
- Symptom Logging -> Story Memory intake
- Timeline -> Thread Timeline
- Graph Explorer -> Thread Evidence Graph
- Missing Data -> Missing Context + Normal-Test Safety Net
- Doctor Summary -> User-controlled Visit Packet
- Document Upload -> Patient-held Record Import

## Reframe

- Hypothesis Engine outputs become questions and relationships to discuss, not diagnosis-like claims.
- Medical institution integration becomes user-controlled import/share first, B2B workflow later only if explicitly chosen.
- Cross-user research becomes opt-in sandbox only, not platform default.

## Add

- Responsibility Memory ledger
- Pending result tracker
- Referral lifecycle tracker
- Post-visit plan checker
- Patient correction loop
- Access and equity memory
- Share Grant system
