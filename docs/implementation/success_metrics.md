# Success metrics

## User value metrics

- percent of active Health Threads with a clear timeline
- percent with patient words captured
- percent with baseline-change field completed
- number of unresolved threads with pending items tracked
- user-reported confidence preparing for visit
- user comprehension of next steps after visit

## Safety and continuity metrics

- pending results/referrals overdue
- pending items with unknown owner/contact
- threads with symptoms persisting after normal tests and no follow-up plan
- repeat-visit/persistence threads surfaced
- correction requests submitted and resolved
- false-positive/false-negative review count

## Product quality metrics

- average Visit Packet length
- source-linked claim coverage
- summary reading time
- share grant revocation rate
- user edits/corrections per generated summary

## Equity metrics

- usage by language/accessibility mode
- completion rates for low-literacy/adaptive forms
- access-barrier fields used
- offline/SMS/print mode success where deployed
