# Low-resource and fragmented-care workflow

This mode is for settings where formal EHR integration, stable primary-care ownership, specialist access, broadband, or reliable diagnostic capacity may be absent.

## Supported modes

- offline thread summary
- printable Health Thread card
- SMS or messaging-based pending item reminders
- photo/PDF/manual import
- caregiver or CHW-assisted intake
- referral slip capture
- stockout/unavailable-test note
- alternative plan tracking

## Guardrails

- do not assign clinical responsibility without local capacity
- show source quality and verification status
- do not blame the user for missed follow-up caused by cost, distance, entitlement, transport, or system delay
- connect CHW mode to local protocols, supervision, supplies, and referral backup before deployment
