# Features to defer or avoid

## Avoid

- Autonomous diagnosis or rare-disease prediction presented directly to the user.
- “AI found what your doctor missed” framing.
- Cross-patient comparison enabled by an institution or on by default.
- Population analytics as a default WellBe output.
- Unscoped clinician/institution access.
- Medication-start/stop advice.

## Defer until clinical safety review

- Bias/misattribution guardrail.
- Decision/uncertainty memory based on clinician reasoning.
- Cross-specialty pattern map.
- Workload-aware clinician worklists.
- CHW mode without local protocol and supervision.

## Reason

These features may be valuable, but they can change clinical behavior, create liability, amplify bias, or turn WellBe into a clinical system rather than a personal memory platform.
