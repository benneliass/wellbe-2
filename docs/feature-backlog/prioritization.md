# Feature prioritization

## High value / lower implementation risk

- Story Memory intake
- baseline change capture
- evidence traceability
- Health Thread timeline
- patient correction loop
- scoped share/export

## High value / higher risk

- Health Thread core
- normal-test safety net
- pending result tracker
- referral lifecycle tracker
- Visit Packet
- Responsibility Memory ledger

## Near-term after MVP

- repeat-visit/persistence view
- missing context checklist
- patient-held record import
- access/equity memory
- wearable trend summarizer
- medication/access clue capture

## Later / caution

- decision and uncertainty memory
- cross-specialty pattern map
- note/document delta view
- care-team comment mode
- doctor discovery as pathway support

## Post-MVP investigation layer (expanded vision)

- Investigation object + Self Ongoing Investigation (P2)
- Theory Evaluator (P5, elevates Myth Buster)
- External Evidence Watch + External Evidence Graph (P6, elevates Research Agent)
- Live Metrics Safety Monitor (P7)
- Clinician Case Investigation Workspace (P3) and Shared Health Thread (P4) — grant-gated
- Full Health Context Summary (P10, expands Visit Packet)

## Deferred — governance gated

- Institution Continuity Intelligence (P8, aggregate-only, consented)
- Research Sandbox / cohort comparison (P9, opt-in, protocol-governed)

## Avoid for MVP

- autonomous diagnosis
- cross-patient comparison
- clinician workload-aware worklists as default product
- institution-enabled aggregate analytics
