# C12/C13 Alignment Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the approved C12 audit/notification and C13 versioned contract slices needed to finish the C1-C6 redesign alignment.

**Architecture:** Add shared Pydantic contracts first, then pure in-memory service/enforcement helpers that can be tested without infrastructure. Expose minimal C13 `/v2` schema/render/audit scaffolding in the API app so the contract boundary exists while deeper persistence can follow behind the same DTOs.

**Tech Stack:** Python 3.13, Pydantic v2, FastAPI, pytest, uv workspace packages.
